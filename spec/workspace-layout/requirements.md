# Workspace Layout Requirements

## User Value

設定を変えながらチャート群を比較しやすくするため、操作系を常に見える位置に置き、分析対象だけをスクロールできるようにする。

## Requirements

- 銘柄入力、期間、DD軸、x軸ズーム、ローソク足、テクニカル指標設定を上部にまとめる。
- 上部設定エリアはデスクトップで固定表示する。
- `Drawdownプロファイル比較` と個別銘柄カードは下部のスクロール領域に置く。
- モバイルでは設定エリアが高くなりすぎないよう、通常スクロールに近い挙動にする。

## Out of Scope

- 設定パネルの折りたたみ。
- チャートカードの仮想スクロール。
- 複数レイアウトプリセット。

## Acceptance Criteria

- デスクトップでチャートをスクロールしても設定系が上部に残る。
- 比較チャートと個別カードだけが縦スクロールする。
- モバイルで操作不能な固定ヘッダーにならない。

# Recovery Analysis Requirements

## User Value

最大drawdownだけでなく、下落の深さ、底打ちまでの日数、回復までの日数を確認し、銘柄ごとの下落耐性と回復力を比較できる。

## Requirements

- 各銘柄について最大drawdownのピーク日、底日、回復日を表示する。
- ピーク日から底日までの暦日数を表示する。
- 底日から回復日までの暦日数を表示する。
- 未回復の場合は `未回復` と回復進捗を表示する。
- 複数銘柄のdrawdownを1枚の比較チャートに重ね描きする。
- 日経平均も比較チャートと個別カードに表示する。
- 歴史的な暴落イベントを赤い点線で表示する。

## Out of Scope

- 営業日数ベースの計算。
- 暴落イベントの外部API連携。
- 回復力指標を使ったランキングやスクリーニング。

## Acceptance Criteria

- `7203, 6758, 9984` で比較チャートと個別回復指標が表示される。
- 未回復銘柄では回復日数ではなく未回復表示になる。
- 暴落イベント線は比較チャートと個別チャートの両方に表示される。

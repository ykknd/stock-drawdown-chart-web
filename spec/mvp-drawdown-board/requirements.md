# MVP Drawdown Board Requirements

## User Value

日本株の複数銘柄について、選択期間内の価格推移とdrawdownを素早く比較できる。

## Requirements

- ユーザーは複数の日本株コードを入力できる。
- `7203` と `7203.T` は同じ銘柄として扱う。
- 日経平均 `^N225` を比較対象として自動表示する。
- 価格系列は調整後ベースで扱う。
- 各銘柄について最大drawdown、現在drawdown、価格系列、drawdown系列を返す。
- 存在しない銘柄が含まれても、成功した銘柄は表示し、失敗銘柄は理由を表示する。
- 銘柄リストと表示設定はブラウザの `localStorage` に保存する。

## Out of Scope

- 認証、課金、DB保存、ユーザー別履歴。
- ポートフォリオ合成、株数、比率、リバランス計算。
- 本番向け有料データプロバイダーへの切り替え。

## Acceptance Criteria

- `7203, 6758, 9984` を入力すると、それぞれのチャートが表示される。
- `.T` あり・なしの入力が重複表示されない。
- 存在しない銘柄を含めても、他の銘柄の表示が継続する。
- 再読み込み後に銘柄リストと主要設定が復元される。
